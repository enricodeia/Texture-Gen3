Help & Documentation

Getting Started

Texture Gen v3 is a powerful tool for creating PBR texture maps from a single base image. This guide will help you make the most of its features.

Basic Workflow:

  1. Upload a texture image using drag & drop or the browse button
  2. Adjust texture settings to your liking
  3. Use the seamless mode to create tileable textures
  4. Export individual maps or all maps as a ZIP file

Features in v3

  • Seamless Textures: Create tileable textures with the interactive seamless mode. Preview the results instantly in the tiled view.
  • Enhanced UV Controls: More precise control over texture projection with separate X/Y repeat, offset, rotation, and scale.
  • UV Editor: Edit UV mapping directly with the UV Editor tab. Reposition, scale, and rotate UV islands.

Keyboard Shortcuts

Space Toggle auto-rotation
E Export all maps as ZIP
C Export Three.js code
Arrow Keys Rotate model

About Texture Gen v3

What is Texture Gen?

Texture Gen is a professional PBR texture generation tool that allows you to create a complete set of texture maps from a single input image. Perfect for game developers, 3D artists, and designers who need high-quality materials quickly.

Features

  • Generate Base Color, Normal, Roughness, Displacement, AO, and Emission maps
  • Live 3D preview
  • Interactive seamless texture creation with real-time preview
  • Advanced UV projection controls
  • Integrated UV editor
  • Export individual maps or complete texture sets
  • Generate Three.js code for your materials
